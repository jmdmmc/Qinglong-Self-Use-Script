🚀 安装部署
方式一：一键拉取仓库（推荐）
进入青龙面板终端，执行以下命令拉取全部脚本：
# 克隆仓库到青龙脚本目录
git clone https://github.com/你的用户名/你的仓库名.git /ql/scripts/qinglong-self

# 赋予执行权限
chmod -R 755 /ql/scripts/qinglong-self
方式二：手动添加订阅
青龙面板 → 订阅管理 → 新建订阅，填写以下信息：
- 订阅链接：https://github.com/你的用户名/你的仓库名
- 分支：main
- 运行模式：定时更新
🔄 脚本更新
进入脚本目录，执行一键更新命令，同步仓库最新脚本：
cd /ql/scripts/qinglong-self && git pull
也可在青龙面板设置定时订阅更新，自动同步最新内容。
📝 使用说明
1. 将仓库拉取至青龙脚本目录后，刷新面板脚本列表即可看到全部文件
2. 根据脚本注释说明，在青龙环境变量中配置对应账号、密钥等参数
3. 手动单次运行测试，确认无报错后，设置定时任务自动执行
4. 脚本内均附带详细注释，新手可根据注释自定义修改参数
